IT Projekt: Self Assessment Tool für Social Media Analytics


Mit diesem Social Media Analytics - Self Assessment Tool möchten wir eine Einschätzung zu Wissen und zur Umsetzung von Social Media Analytics im eigenen Unternehmen ermöglichen.

Bitte beantworten Sie die folgenden Fragen dafür ehrlich und nach bestem Gewissen!

Im Anschluss gibt es eine Auswertung mit Einteilung in Reifegrade. Außerdem bekommen Sie personalisierte Vorschläge, wie sie Ihr Social Media Analytics Wissen verbessern können.


Es gibt folgende Fragetypen, die Sie an den Icons erkennen:

    Multiple Choice Single Choice Single Choice

Der Fragebogen ist in die 3 Gruppen: Prozesse, Technologie und Organisation gegliedert


Sie können aus zwei Fragenbögen wählen:


Start Fragebogen Anfänger (~10 Minuten)
Start Fragebogen Fortgeschritten (~15 Minuten)
